# Animated binary scaffolding screenshots

- [`binary-knowledge-check-scaffold.svg`](./binary-knowledge-check-scaffold.svg) shows the same place-value scaffold in the knowledge check while reading `0101` and making `10` with switches.

The environment could not download the Playwright Chromium binary, so these are checked-in animated SVG captures that mirror the implemented UI states.
